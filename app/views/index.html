Supporting Documents Service

The following prototypes have been created to demonstrate designs for a service that enables claimant documents to be shared through the health assessment process.

Use this service to:

  • view documents stored in the Document Repository Service
  • Manage document lists
  • edit document details
  • comment on documents
  • add documents to claims/referrals
  • link documents to assessment activities

Full MVP

(May 22 by Team Magpie)

This prototype demonstrates functionality planned for the full MVP release. This will include the following functionality:

  • view list of claimant documents
  • view individual documents
  • view document metadata
  • add a document label
  • add a document created date
  • add a comment to document
  • add document to claim(s)
